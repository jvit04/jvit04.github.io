Spacehack 2026: Proyecto Mangroves

Estado: Completado | Periodo: 2026-Ex1

Descripción: Desarrollo de una interfaz de software interactiva unificada que modela dinámicas hidráulicas y ecológicas costeras. El proyecto incluye un simulador de escorrentía terrestre y modelos de biorremediación mediante islas flotantes para la mitigación de inundaciones.

Miembros del Equipo: Josue Alvarez, María Crespín, David Motoche, Domenica Parrales, José Viteri.

Tecnologías:
HTML5 Tailwind CSS JavaScript Chart.js Git
